## Getting Started

First, run the development server:

### First step

```
'npm install' or 'npm i' in root folder
```

### Second step

```
'npm run dev' in root folder
```

## For production mod:

```
'npm start' in root folder
```

### Last step

#### Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
